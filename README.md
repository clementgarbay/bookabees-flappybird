# Flappybird Bookabees

Simple flappy bird like game built with Phaser.

## Installation

### In development run:

```
npm install 
```
or 
```
yarn
```

### For production version just run (devDependencies not needed):

```
npm install --production
```
or 
```
yarn --production
```

## Build

```
npm run build 
```
or 
```
yarn build
```
